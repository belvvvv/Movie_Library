## Описание
Movie Library — это GUI-приложение на Python для хранения информации о фильмах.

Программа позволяет:
- Добавлять фильмы
- Сохранять данные в JSON
- Фильтровать фильмы по жанру и году
- Просматривать фильмы в таблице

## Возможности
- Добавление фильмов
- Проверка корректности данных
- Фильтрация
- Сохранение и загрузка JSON
- Интерфейс на Tkinter

## Проверки
- Год должен быть числом
- Рейтинг от 0 до 10
- Пустые поля запрещены
